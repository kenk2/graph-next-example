# Graph Next Example

A Fullstack App that organizes Todos

# Core Technologies

- NextJS w/ Typescript
- Apollo GraphQL
- PostGresQL w/ Knex ORM
- Material UI

# How to Run

1. Create a `.env` file and add the following: `DATABASE_URL=YOUR_CONNECTION_STRING_HERE`
2. `yarn install`
3. `yarn knex:downup`
4. `yarn dev`

Your app should be ready at http://localhost:3000!
