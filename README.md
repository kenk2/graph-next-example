# Graph Next Example

A Fullstack App that organizes Todos

# Core Technologies

- NextJS w/ Typescript
- Apollo GraphQL
- PostGresQL w/ Knex

# How to Run

1. Create a `.env` file and add the following: `DATABASE_URL=YOUR_CONNECTION_STRING_HERE`
2. `yarn dev`
3. Your app should be ready to go at port `3000`!
